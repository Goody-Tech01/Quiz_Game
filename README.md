# Quiz Game

## Description
This is a simple quiz game built using Python and Tkinter. It presents multiple choice questions loaded from a JSON file and calculates the user's score in real time.

## Features
1. Multiple choice questions
2. Questions loaded from JSON file
3. Next and Previous navigation
4. Retry option
5. Final score display after completion

## Technologies Used
1. Python
2. Tkinter
3. JSON

## How to Run
1. Download or clone the repository
2. Ensure question.json is in the same folder
3. Run the file:
    Quiz_Game.py
